# DoseCerta

## 🎯 Objetivo do Projeto
O DoseCerta é um aplicativo mobile para ajudar pacientes a **lembrar de tomar medicamentos no horário certo**, melhorando a adesão ao tratamento.

---

## 🧩 Problema
Muitas pessoas esquecem de tomar medicamentos corretamente, o que pode prejudicar a saúde e atrasar tratamentos.

---

## 💡 Solução
- Aplicativo simples e intuitivo  
- Permite cadastrar medicamentos e horários  
- Lista os medicamentos e permite marcar como “tomado”  
- (Opcional) Notificações para lembrar o usuário  

---

## 🏗 Estrutura do App

### Telas principais
1. **Home** – Lista de medicamentos  
2. **Cadastro** – Adicionar novo medicamento  
3. **Detalhes** (opcional) – Mais informações do medicamento  

### Tecnologias usadas
- **React Native** – Desenvolvimento mobile  
- **Expo** – Ferramenta de criação e build do app  
- **AsyncStorage** – Armazenamento local  
- **Expo Notifications** – (Opcional) Lembretes de medicação  

---

## ⚙️ Funcionalidades

- **Adicionar medicamento** – Nome e horário  
- **Listar medicamentos** – Visualização fácil  
- **Marcar como tomado** – Sinaliza medicamentos já tomados  
- **Excluir medicamento** – Remove da lista  
- **Notificações (opcional)** – Lembretes automáticos  

---

## 🚀 Como funciona

1. O usuário abre o app → HomeScreen mostra a lista de medicamentos.  
2. Clica em “Adicionar” → AddMedScreen abre formulário para cadastrar medicamento.  
3. Marca como tomado → o app atualiza o status e salva localmente.  
4. Fecha e abre o app → os dados permanecem salvos.  

---

## 🎨 Interface

- Fundo branco e limpo  
- Cards para cada medicamento  
- Botão azul para adicionar  
- Verde = tomado, Vermelho = atrasado  

---

## 📄 Conclusão
O DoseCerta organiza os medicamentos, facilita a rotina do paciente e melhora a adesão ao tratamento.  
Mesmo simples, é **útil, funcional e impressiona** pelo cuidado com a experiência do usuário.
